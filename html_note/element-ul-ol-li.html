ul/ol/li要素

ul
説明 順不同のリストを生成。ul要素の直下にはul要素を置けないが、li要素の子要素として入れ子にすることは可能。
使用できる属性 グローバル属性

ol
説明 序列リストを生成。ol要素の直下にはol要素を置けないが、li要素の子要素として入れ子にすることは可能。
使用できる属性 グローバル属性, reversed, start, type

li
説明 ul要素やol要素の子要素として、文章をリスト化する。
使用できる属性 グローバル属性, value

コード例

#順不同リストの作成
<ul>
    <li>リスト1<>
    <li>リスト2<>
    <li>リスト3<>
<ul>
#開始
  • リスト1
  • リスト2
  • リスト3
#終了 
#リストの入れ子(順不同)
<ul>
    <li>買い物リスト
        <ul>
            <li>牛乳</li>
            <li>牛肉</li>
            <li>卵</li>
        </ul>
    </li>
    <li>やることリスト
        <ul>
            <li>掃除</li>
            <li>買い物</li>
            <li>犬の散歩</li>
        </ul>
    </li>
</ul>
#開始
  • 買い物リスト
    • 牛乳
    • 牛肉
    • 卵
  • 買い物リスト
    • 掃除
    • 買い物
    • 犬の散歩
#終了 
#序列リストの作成
<ol>
    <li>鬼滅の刃</li>
    <li>進撃の巨人</li>
    <li>呪術廻戦</li>
</ol>
#開始
  1. 鬼滅の刃
  2. 進撃の巨人
  3. 呪術廻戦
#終了
