ul/ol/li要素

ul
説明 順不同のリストを生成。ul要素の直下にはul要素を置けないが、li要素の子要素として入れ子にすることは可能。
使用できる属性 グローバル属性

ol
説明 序列リストを生成。ol要素の直下にはol要素を置けないが、li要素の子要素として入れ子にすることは可能。
使用できる属性 グローバル属性, reversed, start, type

li
説明 ul要素やol要素の子要素として、文章をリスト化する。
使用できる属性 グローバル属性, value

コード例

・順序なしリストの作成 
<ul>
    <li>リスト1<>
    <li>リスト2<>
    <li>リスト3<>
<ul>
  • リスト1
  • リスト2
  • リスト3


・リストの入れ子(順不同) 
<ul>
    <li>買い物リスト
        <ul>
            <li>牛乳</li>
            <li>牛肉</li>
            <li>卵</li>
        </ul>
    </li>
    <li>やることリスト
        <ul>
            <li>掃除</li>
            <li>買い物</li>
            <li>犬の散歩</li>
        </ul>
    </li>
</ul>
  • 買い物リスト
    • 牛乳
    • 牛肉
    • 卵
  • 買い物リスト
    • 掃除
    • 買い物
    • 犬の散歩


・序列リストの作成 
<ol>
    <li>鬼滅の刃</li>
    <li>進撃の巨人</li>
    <li>呪術廻戦</li>
</ol>
  1. 鬼滅の刃
  2. 進撃の巨人
  3. 呪術廻戦


・olのreversed属性。順序を逆にする。 
<ol reversed>
    <li>鬼滅の刃</li>
    <li>進撃の巨人</li>
    <li>呪術廻戦</li>
</ol>
  1. 鬼滅の刃
  2. 進撃の巨人
  3. 呪術廻戦


・olのstart属性。任意の番号から開始する。 
<ol start="3">
    <li>鬼滅の刃</li>
    <li>進撃の巨人</li>
    <li>呪術廻戦</li>
</ol>
  1. 鬼滅の刃
  2. 進撃の巨人
  3. 呪術廻戦


・olのtype属性。リスト番号の種類を指定可能。 
<ol type="a">
    <li>鬼滅の刃</li>
    <li>進撃の巨人</li>
    <li>呪術廻戦</li>
</ol>
