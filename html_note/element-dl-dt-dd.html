dl/dt/dd要素

dl
説明 説明リストの作成。語句に対する説明を記述するブロック。
使用できる属性 グローバル属性

dt
説明 説明リストの説明される語句。
使用できる属性 グローバル属性

dd
説明 説明リストの語句に対する説明文。
使用できる属性 グローバル属性

コード例

・説明リスト 
<dl>
    <dt>C言語</dt>
    <dd>デニス・リッチーが主体となって開発した汎用プログラミング言語。</dd>
    <dt>Python</dt>
    <dd>
        インタープリター型の高水準汎用プログラミング言語。<br />
        ホワイトスペースの使用によってコードの可読性を重視している。
    </dd>
</dl>
C言語

Python
インタープリター型の高水準汎用プログラミング言語。
ホワイトスペースの使用によってコードの可読性を重視している。
