ins/del要素

ins
説明 文章に後から追記、挿入されたテキストを表す。
使用できる属性 グローバル属性, cite, datetime

del
説明 文章から削除されたテキストを表す。
使用できる属性 グローバル属性, cite, datetime

・文章の改行 
<h2>ToDo リスト</h2>
<ul>
    <li>部屋の掃除</li>
    <li><del>牛乳を買う。</del></li>
    <li><ins>課題の提出</ins></li>
    <li><ins>ゼミの予習</ins></li>
</ul>

ToDo リスト

  • 部屋の掃除
  • 牛乳を買う。
  • 課題の提出
  • ゼミの予習


・cite要素(URLでリソースを指定) 

    <h2>ToDo リスト</h2>
    <ul>
    <li>部屋の掃除</li>
    <li><del cite="https://www.google.com/?hl=ja">牛乳を買う。</del></li>
    <li><ins>課題の提出</ins></li>
    <li><ins cite="https://www.google.com/?hl=ja">ゼミの予習</ins></li>
    </ul>

ToDo リスト

  • 部屋の掃除
  • 牛乳を買う。
  • 課題の提出
  • ゼミの予習


・datetime要素(追加、削除された日時を表す) 

    <h2>ToDo リスト</h2>
    <ul>
    <li>部屋の掃除</li>
    <li><del datetime="2020-01-23">牛乳を買う。</del></li>
    <li><ins>課題の提出</ins></li>
    <li><ins datetime="2020-01-24">ゼミの予習</ins></li>
    </ul>

ToDo リスト

  • 部屋の掃除
  • 牛乳を買う。
  • 課題の提出
  • ゼミの予習
